# XRD Structure Matching Skill

## overview

Identify a crystalline phase from an experimental X-ray diffraction (XRD)
pattern by matching against database structures. The skill provides three
tools — `search_structures`, `simulate_xrd_pattern`, `score_xrd_match` —
that the analysis script chains together to query candidates, simulate their
kinematic patterns, and score against the experimental data.
